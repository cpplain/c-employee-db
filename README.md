# C Employee Database

A file-based employee database written in C.

## Usage

Build from source and test:

```shell
make
make run-srv
make run-cli
```

Print server options:

```shell
build/bin/dbsrv -h
```

Print CLI options:

```shell
build/bin/dbcli -h
```
